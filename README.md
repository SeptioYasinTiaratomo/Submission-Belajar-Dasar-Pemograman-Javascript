# Submission: Belajar Dasar Pemrograman JavaScript

## 📖 Deskripsi
Repositori ini berisi kode sumber untuk *submission* kelas **Belajar Dasar Pemrograman JavaScript**. Proyek ini dibuat untuk memenuhi kriteria kelulusan kelas dan mendemonstrasikan pemahaman praktis mengenai sintaks JavaScript, manipulasi struktur data, konsep *Object-Oriented Programming* (OOP), serta kebiasaan penulisan kode yang bersih (*clean code*).

## 🚀 Fitur & Materi yang Diimplementasikan
Berdasarkan silabus pembelajaran, proyek ini mencakup implementasi:
- **Fundamental JavaScript:** Penggunaan variabel, tipe data, *statement*, dan *expression*.
- **Struktur Data:** Manipulasi *Array*, *Object*, *Map*, dan *Set*.
- **Fungsi:** Deklarasi fungsi, penanganan parameter, dan penerapan konsep *Pure Function*.
- **Object-Oriented Programming (OOP):** Pembuatan *Class*, properti, dan *method*.
- **Asynchronous Process:** Penanganan proses *async* menggunakan *Promise*, `setTimeout`, atau `async/await`.
- **Modularisasi:** Menerapkan proses ekspor dan impor modul (CommonJS / ES Modules).
- **Unit Testing:** Menguji fungsionalitas kode menggunakan pola AAA (*Arrange-Act-Assert*).
- **Linter:** Penggunaan ESLint untuk menjaga konsistensi gaya kode.

## 🛠️ Teknologi yang Digunakan
- **Bahasa:** JavaScript (ES6+)
- **Runtime Environment:** Node.js (atau Bun)
- **Testing Framework:** Jest (atau menyesuaikan materi kelas)
- **Linter:** ESLint
